# ChattyExpressMongo

Hello guys or gals, this is an app that allows users to chat with other people

What does this use?

* Express
* MongoDB (mongoose as the ODM)
* React
* Passport

To run this app, you must

```bash
npm install
```
or
```bash
yarn install
```


after you have installed all of the dependencies you can


```bash
npm start
```
or
```bash
yarn run start
```
