# ChattyExpressMongo

This basicly allows users to be able to talk to each other

It uses:

* Express
* MongoDB (mongoose as the ODM)
* React
* MobX
* Passport
* JWT
* Socket.io

To run this app, you must

```bash
yarn install
```


After you install everything, you run

```bash
yarn build
```
Then go and


```bash
yarn start
```
